# Kois

Mobile app school project to explore and research a specific area on the map

## Getting Started

> Download the project<br/>
> npm install<br/>
> Setup expo on your pc && phone </br>
> that's it... now just run "npm run start"<br/>

### Prerequisites

* NodeJS
* Expo

## Built With

* reactNative
* some random js libraries
